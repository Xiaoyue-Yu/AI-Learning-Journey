## Quick start

1. `python -m venv .venv` then activate it.
2. `pip install -r requirements.txt`
3. Copy `.env.example` to `.env` and set `OPENROUTER_API_KEY`.
4. Run `python 02_OpenRouter_Test.py` or `03_Structured_Output.py`.
